The one idea: When a model reads text, at every layer it holds a list of numbers (a vector) for each token. That internal vector is called an activation, and the running stack of them through the layers is the residual stream. Everything we do later — steering, SAEs — is just reading and nudging these vectors. So step one is simply to grab one and look at it.

The one idea: if happy sentences live in one region of activation space and sad ones in another, then the arrow from the sad cluster to the happy cluster is a "happiness direction." We build it the simplest way possible: average the activations of some happy sentences, average the sad ones, and subtract. No training, no library — just a subtraction.
